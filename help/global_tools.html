Global tools

Global tools toolbar provides quick access to frequently used actions of the Main menu.

  • – create a new work selecting a type of model to capture.
  • – load an existing work from a file.
  • – write the current work into a file.
  • – undo the last modification in the current work.
  • – redo a previously undone modification in the current work.
  • – copy selected nodes and connections into a local clipboard.
  • – paste a nodes and connections from the local clipboard into the current work.
  • – zoom in the current work by one step.
  • – zoom out the current work by one step.
  • – restore the default scale of the current work.
  • – scale and pan the current work so that its selected objects (or the whole model if nothing is selected) fill the editor panel.
  • – toggle grid visibility.
  • – toggle ruler visibility.
  • – toggle name visibility.
  • – toggle label visibility.
