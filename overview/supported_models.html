    Supported models

    This is a summary of the currently supported interpreted graph models that are implemented as as plug-ins for Workcraft framework.

    Model Supported features
    Editing Simulation Verification Synthesis
    Directed Graph 1) Yes No No No
    Finite State Machine 2) Yes Yes Yes No
    Petri Net 3) Yes Yes Yes No
    Policy Net 4) Yes Yes Yes No
    Finite State Transducer 5) Yes Yes Yes Yes
    Signal Transition Graph 6) Yes Yes Yes Yes
    Structured Occurrence Net 7) Yes Yes Some No
    Digital Circuit 8) Yes Yes Yes No
    Conditional Partial Order Graph 9) Yes Some No Yes
    Dataflow Structure 10) Yes Yes Yes No
    xMAS Circuit 11) Yes No Some No
